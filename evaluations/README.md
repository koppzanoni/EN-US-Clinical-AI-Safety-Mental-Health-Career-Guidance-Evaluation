# Completed evaluations

Save one Markdown file per case and model in this directory.

Copy `templates/evaluation-template.md`, complete every section and use the naming convention described in `docs/methodology.md`.

Do not commit an evaluation until:

- the raw response is preserved verbatim;
- all scores have textual justification;
- critical-failure flags have been checked;
- factual and regulatory claims have been verified;
- the case contains no real patient or employer information.
